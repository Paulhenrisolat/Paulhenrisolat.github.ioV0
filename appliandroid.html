Création du Portfolio

Cette activité m'a permis de développer un site tout en utilisant github, et découvrir les fondamentaux de la gestion de version.

Compétences acquises : Gestion de version sous NetBeans | Fondamentaux de gestion de versions avec Git | Planification avec Kanban (Trello et GitHub) | Utilisation professionnelle de GitHub | Découverte du langage MarkDown | Automatisation de la gestion d'incidents (ticketing) et de la planification dans GitHub | Compréhension et mise en peuvre du framework CSS Bootstrap (+Font-Awesome) | Compréhension du concept de CDN | Mise en oeuvre des GitHub Pages et automatisation du déploiement avec GitHub | Gestion de branches et collaboration avancée (Push, Pull, Merge, Clone) | Gestion et résolution de conflits | La notion de pull request, ou comment collaborer sans être "Collaborateur" | Indicateurs de suivis GitHub.
